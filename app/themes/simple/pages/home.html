Welcome to Galactic Mango CMS!

This is the starter theme for Galactic Mango CMS. Galactic Mango CMS is a responsive content management system built in Angular that you can use to build responsive websites. Use it as a starting point to create your own site.

Better with Bootstrap

Respond is built on Twitter Bootstrap 3.0. This means the sites you create using Galactic Mango CMS are built on a proven, powerful framework that is easy to extend and adapt to your needs. Building your site has never been easier with Galactic Mango CMS and Bootstrap.
